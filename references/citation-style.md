# Citation Style

Use these formats consistently across analysis and reporting.

## Quantitative citation

Preferred pattern:

```text
[metric or result] ([source or question], n=[sample size], [definition or test note])
```

Example:

```text
Borrowing intent was 28.5% (Survey Q4, n=1,920, excluding invalid responses)
```

## Qualitative quote citation

Preferred pattern:

```text
"[quote]" — Interview#[id] | [profile] | [question or topic]
```

Do not use anonymous placeholders like `User A` when traceability matters.

## Code / pattern citation

Preferred pattern:

```text
[pattern or code] ([count]/[base], [percent]%)
```

## General rules

- Keep source references stable across the report.
- Define specialized metrics the first time they appear.
- If quotes were lightly edited for readability, say so once in the methods or note.
